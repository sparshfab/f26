Software

For me chosing the cad software for my Fab Academy was easy. I have never worked with 2D CAD, I had heard a lot about incscape for vector work so i have decided to use that. Cad Project

For Learning Fusion 360 I decided to design a Peristaltic Pump that uses a motor i salveged from an old vaccum robot. It uses 5mm OD silicone tube and has 3 roller design. ALl the parts are to be 3d printable and the design is parametric to make the tolerence adjustment easy.
